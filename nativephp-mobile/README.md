# NativePHP Mobile Plugin

A comprehensive plugin for building iOS and Android applications with NativePHP Mobile.

## Features

### Skills

- **nativephp-quickstart** - New project setup, environment configuration, installation
- **nativephp-apis** - All native APIs (Camera, Device, Dialog, Scanner, Biometrics, etc.)
- **nativephp-events** - Event system for native-to-app communication
- **nativephp-edge** - EDGE components (TopBar, BottomNav, SideNav, Fab)
- **nativephp-config** - Permissions, configuration options, environment variables
- **nativephp-deployment** - Build, sign, package, and submit to app stores
- **nativephp-livewire** - Livewire-specific patterns and event handling
- **nativephp-javascript** - JavaScript/Vue/React patterns and imports

### Agents

- **nativephp-livewire-validator** - Validates Livewire components using NativePHP
- **nativephp-js-validator** - Validates JavaScript/Vue/React NativePHP code
- **nativephp-config-validator** - Validates configuration and permissions

## Installation

```bash
claude --plugin-dir /path/to/nativephp-mobile
```

Or add to your project's `.claude/plugins/` directory.

## Documentation

All skills fetch live documentation from https://nativephp.com/docs/mobile/2/

## Requirements

- NativePHP Mobile license
- PHP 8.3+
- Laravel 11+
- Xcode 16+ (iOS) and/or Android Studio 2024.2.1+ (Android)
